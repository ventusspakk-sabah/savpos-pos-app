App Wrapping Guide.

Transform your Sav.pos web experience into a native iOS and Android application using Capacitor.

terminal 1. Environment Setup

Ensure your development machine is ready for native builds.

  • Node.js: Install the latest LTS version from nodejs.org.
  • Android Studio: Required for Android builds. Install Command Line Tools.
  • Xcode: Required for iOS builds (macOS only).

download 2. Code Export

Download your project files from the Stitch platform.

1. Navigate to Project Settings
2. Select 'Export Production Build'
3. Unzip the downloaded folder into your project directory.

rocket_launch 3. Project Initialization

Initialize Capacitor in your unzipped project folder.

npm install @capacitor/core @capacitor/cli
npx cap init "Sav.pos" "com.savpos.app" --web-dir dist

add_to_home_screen 4. Platform Addition

Add the native platforms you want to target.

npm install @capacitor/ios @capacitor/android
npx cap add ios
npx cap add android

sync 5. Building & Syncing

Sync your web code with the native project files.

npm run build
npx cap sync

play_circle 6. Final Launch

Open the project in the respective IDE to run on a device or emulator.

npx cap open ios
npx cap open android
