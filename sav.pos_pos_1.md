# Sav.pos POS 總機系統功能走查與強化清單

## 1. 房間與現場調度 (總機指揮中心)
- [x] **實時樓層概覽**：動態顯示所有房間狀態（閒置、服務中、待清潔）。
- [x] **快捷開單流程**：點擊房間直接進入服務選擇。
- [x] **技師智能調度**：支持手動指定或根據技師狀態自動分配，解決「算人頭」與排班衝突。
- [x] **服務進度監測**：全螢幕進度條與超時預警，確保服務品質。

## 2. 收銀與財務閉環 (核心帳務端)
- [x] **多重支付 (Split Payment)**：支持一筆訂單使用多種支付方式（現金、卡片、積分）。
- [x] **自動抹零設定**：自定義抹零規則，優化結帳效率。
- [x] **掛單/取單功能**：繁忙時段的訂單暫存與快速喚回。
- [x] **退款與作廢流程**：嚴格的角色權限控制，連動佣金自動沖銷邏輯。

## 3. 會員與營銷引擎
- [x] **積分體系整合**：結帳時自動累計積分，支持積分抵扣。
- [x] **優惠券核銷**：掃碼或輸入代碼快速套用促銷折扣。
- [x] **會員等級權益**：自動識別身份並套用專屬折扣率。

## 4. 系統基礎設施與硬體
- [x] **日終結帳 (Z-Report)**：交接班時的財務對帳與業績彙總。
- [x] **硬體狀態監控**：即時監測印表機、掃描槍等設備連接狀態。
- [x] **雲端數據同步**：與 Google Drive 連動，確保總機數據永不丟失。
