# Sav.pos 完整原始碼下載與部署指南

您的 Sav.pos 系統已正式開發完成並封裝為三個獨立的應用程式模組。請參考以下步驟獲取代碼並將其上線營運。

## 📦 如何下載原始碼 ZIP 包
1. 在 Stitch 介面右上角點擊 **「View Code ⟨/⟩」** 圖示。
2. 在開啟的代碼視窗中，點擊右上角的 **「Download ZIP」** 按鈕。
3. 系統將會生成並下載名為 `savpos_full_source.zip` 的壓縮檔到您的電腦。

## 📁 資料夾結構說明
解壓 ZIP 包後，您將看到以下三個獨立的專案目錄：

- **`/savpos-pos-app` (🧖 前台總機系統)**
  - 核心功能：房控分配、收銀結帳、授權調價、Z-Report 報表。
  - 適用對象：前台接待與收銀人員。

- **`/savpos-tech-app` (💆 技師個人端)**
  - 核心功能：佣金明細、30天帳期保護、服務開始/結束、客戶簽名、報銷申請。
  - 適用對象：按摩師與技師。

- **`/savpos-admin-app` (👔 老闆管理端)**
  - 核心功能：業績看板、審批中心、權限矩陣、Google Drive 備份與數據核對。
  - 適用對象：店主與財務管理。

## 🚀 快速部署步驟 (Vercel / Netlify)
如果您想要獲得免費網址並在手機上使用，建議使用 Vercel 或 Netlify：

1. **上傳至 GitHub**：將上述三個資料夾分別作為獨立的倉庫 (Repository) 上傳至 GitHub。
2. **連結部署平台**：登入 Vercel 或 Netlify，選擇「Import from GitHub」。
3. **自動生成網址**：平台會自動偵測 `package.json` 並完成構建，您將獲得三個獨立的 `.vercel.app` 網址。
4. **手機安裝 (PWA)**：在手機瀏覽器開啟網址，點擊「加入主畫面」即可像 App 一樣使用。

## 🛠️ 開發者環境啟動
如果您想在本地電腦執行：
1. 開啟終端機 (Terminal) 並進入對應的應用程式資料夾。
2. 執行 `npm install` 安裝必要組件。
3. 執行 `npm run dev` 啟動預覽。
