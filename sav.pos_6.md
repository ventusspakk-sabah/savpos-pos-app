# Sav.pos 全系統功能交付清單 (Final Handover Checklist)

## 1. 核心 POS 收銀端 (POS Application)
- [x] **收銀與結帳**：卡片式服務目錄、購物車、多重支付 (Split Payment)、自動抹零。
- [x] **房間與現場管理**：樓層概覽 (Floor Overview)、動態房間狀態 (呼吸燈效果)、服務監測。
- [x] **技師調度**：手動/自動分配技師、服務開始/結束控制。
- [x] **財務與結算**：日終結帳報表 (Z-Report)、訂單作廢與退款流程。

## 2. 技師端應用程序 (Technician Application)
- [x] **個人看板**：即時佣金明細、30 天帳期保護邏輯、工作通知。
- [x] **服務執行**：服務小結、客戶簽名確認。
- [x] **申請中心**：報銷申請 (含超額阻斷提示)、提現申請流程。

## 3. 老闆/管理員後台 (Admin & Management)
- [x] **數據儀表板**：業績排行、營收趨勢圖表、月度對比分析。
- [x] **權限控管**：4 位數 PIN 碼安全解鎖、RBAC 角色與權限矩陣。
- [x] **系統基礎設施**：Google Drive 數據備份與還原、數據核對中心 (Reconciliation Center)。
- [x] **第三方整合**：Google 商家、地圖、日曆等服務整合中心。

## 4. 線上預約門戶 (Public Booking Portal)
- [x] **網站生成器**：SEO 自動填充 (AI 建議)、網站模板選擇、域名綁定介面。
- [x] **客戶體驗**：服務詳細介紹、熱門推薦、優惠券核銷、預約成功確認、數位收據。
- [x] **客戶個人中心**：預約歷史管理、個人資料編輯、白金會員進度追蹤。
