# Sav.pos Project Handover Documentation

## 1. Project Overview
Sav.pos is a comprehensive, end-to-end POS and management solution specifically designed for the spa and wellness industry. The system bridges the gap between front-desk operations, technician workflows, and executive management.

## 2. Core Application Modules

### 🧖 Front-Desk POS Application
- **Sales & Checkout**: High-performance product grid, multi-payment (Split Payment) support, automated rounding, and hold/resume order management.
- **Room Control**: Real-time floor overview with room status (Available, Serving, Cleaning) and service progress monitoring.
- **Appointments**: Integrated booking calendar with conversion flow to active orders.

### 💆 Technician Application
- **Earnings & Commission**: Transparent real-time commission tracking with a 30-day held period protection and clawback logic for refunds.
- **Service Operations**: Instant notifications for assigned jobs and streamlined service start/stop controls.
- **Requests**: Standardized expense reimbursement and leave request forms with built-in policy enforcement (over-limit blocking).

### 👔 Executive/Admin Application
- **Insights Dashboard**: Real-time performance analytics, revenue trends, and staff productivity rankings.
- **System Management**: Granular RBAC (Role-Based Access Control) with PIN-code security, service catalog configuration, and automated notification templates.
- **Financial Audit**: Batch payout tools, Z-Report previews, and comprehensive audit logs for compliance.

## 3. Technical Specifications
- **Security**: 4-digit PIN-code access, role-based permissions, and full audit trailing.
- **Integrations**: Pre-configured for thermal printing, barcode scanning, and multi-store data synchronization.
- **Robustness**: Verified via high-concurrency stress testing for calculation integrity and data consistency.

## 4. Visual Asset Directory
The project includes over 100 high-fidelity mobile screens, all unified under the **Sav.pos Professional** design system (Inter typography, #002045 brand primary).
