# Sav.pos PWA (Progressive Web App) 部署指南

Sav.pos 系統已內建 PWA 支援，讓您無需通過 App Store 即可在手機上安裝並獲得原生 App 的操作體驗。

## 1. 什麼是 PWA？
PWA 是一種 Web 技術，它讓網頁應用程序具備以下特性：
- **可安裝性**：可以將圖示添加到手機主畫面，像原生 App 一樣啟動。
- **全螢幕運行**：隱藏瀏覽器網址列與操作按鈕，提供沉浸式體驗。
- **離線支援**：具備基本的離線運行能力。
- **高性能**：快取核心資源，啟動速度極快。

## 2. iPhone (iOS) 安裝步驟
1. 使用 **Safari** 瀏覽器開啟 Sav.pos 系統網址。
2. 點擊瀏覽器下方工具列正中央的 **「分享 (Share)」** 圖示（一個向上箭頭的方框）。
3. 在彈出的選項清單中，向下滑動並點擊 **「加入主畫面 (Add to Home Screen)」**。
4. 確認名稱為「Sav.pos」，點擊右上角的 **「新增」**。
5. 現在，您可以回到主畫面，點擊蓮花圖示啟動應用。

## 3. Android 安裝步驟
1. 使用 **Chrome** 瀏覽器開啟 Sav.pos 系統網址。
2. 點擊網址列右側的 **「三個點」** 選單。
3. 點擊 **「安裝應用程式 (Install App)」** 或 **「新增至主畫面」**。
4. 在彈出的確認視窗中點擊 **「安裝」**。
5. 稍候片刻，Sav.pos 圖示將出現在您的應用程式清單或主畫面上。
